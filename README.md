# 牌効率チェッカー

## About (概要)

https://github.com/nekobean/mahjong-calculator を emscripten対応し、牌効率チェッカー用に改変したものです

emscripen化対応コードは https://github.com/Muratam/mahjong-cpp/ にあります。
